# Dockerfiles for [php-laravel](https://hub.docker.com/repository/docker/sensasidelight/php-laravel) image

PHP Versions:

- [x] 7.4-fpm
- [ ] 7.4-fpm-alpine
- [x] 8.0-fpm
- [ ] 8.0-fpm-alpine
- [x] 8.1-fpm
- [ ] 8.1-fpm-alpine

## Build for yourself

```bash
docker build -f "[version tag].dockerfile" -t "sensasidelight/php-laravel:[version tag]" .
```
